Սկսզբից սկսեցի գուգլել ինֆո որոնել, հետո հասկացա որ կոնկրետ բանի մասին չեմ ինֆո ման գալիս ու սկսեցի յութուբում նմանատիպ վիդեոներ նայել
Վիդեոներից շատ բաներ փորձեցի չստացվեց դրանց միացումից ինչ-որ բան ստանա։
Path երը ուսումնասիրեցի, բայց չհասկացա որը կարող է կոնկրետ պետք գալ մեր խնդրի լուծման մեջ։ Հետո հասկացա, որ ինձ պետքա մի կոդ, որը սլեշից հետո կատարվելիքը պետքա անի։ Ու մինչև վերջ դրա վրա էի տանջվում, բայց չկարողացա։
